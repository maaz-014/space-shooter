# 🚀 Space Shooter

A 2D space shooter game built with **C++** and **SFML 3**, developed as an Object-Oriented Programming (OOP) course project.

## 🎮 Features
- Player-controlled spaceship with keyboard input (WASD / Arrow keys)
- Enemy spawning system with configurable AI firing behavior
- Bullet collision detection system
- Health reward power-ups that drop periodically
- Animated intro screen with fade transition
- Game Over screen with final score display
- Full audio: background music, fire sound, hit sound, reward sound
- Score tracking in real-time HUD

## 🛠 Built With
- **C++17**
- **SFML 3** (Simple and Fast Multimedia Library)
- **Visual Studio 2022**

## 📋 Prerequisites
- [SFML 3](https://www.sfml-dev.org/download/) installed and linked
- Visual Studio 2022 (or compatible)

## ▶️ How to Run
1. Clone the repository
2. Open `oop project.sln` in Visual Studio
3. Link SFML libraries in project properties
4. Build and run in **x64 Debug** mode

## 📁 Project Structure
- `oop project.cpp` — All game logic (single-file architecture)
- `resources/images/` — Sprites and backgrounds
- `resources/Audios/` — Sound effects and music
- `resources/fonts/` — Custom game font (Audiowide)

## 🎓 Academic Context
Developed as a semester project for the **Object-Oriented Programming** course.
